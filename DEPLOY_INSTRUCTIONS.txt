The HTML tool doesn't directly execute SQL, so we need to use Supabase SQL Editor.

┌──────────────────────────────────────────────────────────────────────┐
│  STEP 1: Open Supabase SQL Editor                                   │
└──────────────────────────────────────────────────────────────────────┘

1. Go to: https://supabase.com/dashboard
2. Select your project: cddoboboxeangripcqrn
3. Click "SQL Editor" in the left sidebar
4. Click "New Query" button


┌──────────────────────────────────────────────────────────────────────┐
│  STEP 2: Run First Migration (Add Columns & RLS)                    │
└──────────────────────────────────────────────────────────────────────┘

1. Open file: RUN_THIS_IN_SUPABASE_SQL_EDITOR.sql
2. Copy ENTIRE contents (Ctrl+A, Ctrl+C)
3. Paste into SQL Editor (Ctrl+V)
4. Click "Run" button (or press Ctrl+Enter)
5. Wait for success message
6. You should see:
   ✅ Success. No rows returned
   ✅ Migration 028 completed successfully!


┌──────────────────────────────────────────────────────────────────────┐
│  STEP 3: Run Second Migration (Update Function)                     │
└──────────────────────────────────────────────────────────────────────┘

1. Clear the SQL Editor (select all and delete)
2. Open file: RUN_THIS_SECOND_IN_SUPABASE_SQL_EDITOR.sql
3. Copy ENTIRE contents (Ctrl+A, Ctrl+C)
4. Paste into SQL Editor (Ctrl+V)
5. Click "Run" button (or press Ctrl+Enter)
6. Wait for success message
7. You should see:
   ✅ Success. No rows returned
   ✅ Migration 029 completed successfully!


┌──────────────────────────────────────────────────────────────────────┐
│  STEP 4: Verify in Your App                                         │
└──────────────────────────────────────────────────────────────────────┘

1. Open your app: http://localhost:5173 (or your app URL)
2. Log in as Super Admin
3. Go to Businesses page
4. Click "View" on LatexFoam (or any business)
5. Check Overview tab:
   ✅ Should see "Business ID: xxxxx-xxxx-xxxx..." at top
6. Click "Business Details" tab:
   ✅ Should see highlighted Business ID card at top
   ✅ Should see all business details sections below
